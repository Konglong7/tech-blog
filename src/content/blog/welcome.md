## 你好，世界！

欢迎来到我的技术博客！这是我使用 Astro 搭建的个人技术博客网站。

### 为什么要写博客？

写博客有很多好处：

1. **记录学习过程** - 通过写作整理思路，加深理解
2. **分享技术经验** - 帮助遇到类似问题的开发者
3. **建立个人品牌** - 展示技术能力和思考深度

### 技术栈

这个博客使用了以下技术：

- **Astro** - 现代化的静态网站生成器
- **Tailwind CSS** - 实用优先的 CSS 框架
- **Tina CMS** - 可视化内容管理系统
- **Giscus** - 基于 GitHub Discussions 的评论系统
